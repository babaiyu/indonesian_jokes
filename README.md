# indonesian_jokes
Kumpulan Jokes berbahasa Indonesia
Project ini dibuat untuk memeriahkan hacktoberfest2021

## Cara running di local
```
git clone https://github.com/babaiyu/indonesian_jokes.git
cd indonesian_jokes
npm run install
npm run generate
npm run dev
```
